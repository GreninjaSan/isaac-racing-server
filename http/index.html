Racing+

A racing mod for The Binding of Isaac: Afterbirth+.

  • Download
  • Learn More

Race other Isaac players head to head.

It's free, and it couldn't be easier. Race your friends or make new ones.

Integrated Racing

Everything is integrated within the game. You can play a 1v1 or race a bunch of people at the same time. Streaming isn't required.

Statistics & Leaderboards

The server will keep track of all of your races and times. Compete with others to get the lowest average time. Unlike the vanilla leaderboards, cheaters will be banned.

1P

Single-Player & Practice

If no-one else is around, you can always start a race by yourself. Or, if you need to get better at using a specific item, use the practice mode to start a run with that item.

Bug Fixes

Enjoy an updated client that includes hundreds of bug fixes that the developers have not bothered to fix in the vanilla game.
